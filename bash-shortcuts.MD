## Terminal Shortcuts
Note that some commands only work on the terminal app i.e does not work in an in-app terminal like vscode,..

### Move to beginning of line
`CTRL + A`

### Move to end of line
`CTRL + E`

### Remove last word 
`CTRL + W`

### Remove Line backward 
`CTRL + U`

### Remove Line Forward 
`CTRL + K`

### Undo delete (last word only)
`CTRL + Y`
